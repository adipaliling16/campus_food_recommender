# Campus Food Recommender

Project Flutter siap pakai untuk tugas besar APB: aplikasi rekomendasi tempat makan dan cafe di sekitar kampus berbasis rating dan ulasan pengguna.

## Fitur

- Beranda rekomendasi tempat makan/cafe
- Pencarian berdasarkan nama, menu, lokasi, atau kata kunci
- Filter rating tertinggi dan kisaran harga
- Detail tempat makan lengkap dengan foto, menu, harga, fasilitas, rating, dan ulasan
- Tambah rating dan ulasan pengguna
- Galeri foto sederhana
- Halaman admin untuk tambah, ubah, dan hapus data tempat makan
- Data lokal/in-memory sehingga langsung bisa dijalankan tanpa backend

## Cara Menjalankan

```bash
flutter pub get
flutter run
```

## Struktur Folder

```text
lib/
  main.dart
  models/
  data/
  screens/
  widgets/
```

## Catatan

Project ini dibuat sebagai versi siap presentasi/prototipe. Untuk produksi, data lokal bisa diganti dengan Firebase, Supabase, Laravel API, atau backend lain.
